
This module investigates how contextual patient narratives can enrich traditional clinical datasets.

Key Contributions:
- Designed a bi-directional relational schema connecting clinical symptoms with narrative patient experiences
- Integrated qualitative "thick data" with structured clinical datasets
- Normalized over 250,000 relational rows using Python
- Implemented high-speed ingestion using SQL LOAD DATA
- Resolved encoding conflicts using UTF8MB4 and REGEXP sanitization

Research Impact:
The resulting dashboard allows researchers to pivot from statistical frequency analysis to human behavioral context in patient data.

Example Insight:
Understanding not only how many patients experience thirst but also the social stigma and emotional narratives surrounding symptoms.
