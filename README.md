
# Health Informatics Analytics

This repository presents a structured portfolio of two complementary data analytics projects focused on healthcare data. The work combines statistical analysis, relational database design, and visual analytics to extract meaningful insights from large-scale clinical datasets.

The projects demonstrate the integration of Python, SQL, and Power BI in building end-to-end data pipelines for health informatics research.

## 📌 Projects Overview

### 1. Thin + Thick Data Integration
This project combines large-scale clinical data (thin data) with qualitative patient narratives (thick data) to provide a more complete understanding of disease patterns.

- Focus: Diabetes case study  
- Approach: Mixed-methods analytics  
- Tools: Python, MySQL, Power BI  
- Key Contribution: Bridging statistical trends with human context  

📁 Folder: `Thin_Thick_Integration/`

---

### 2. Graph Logic & Relational Topology
This project applies graph-based thinking to model relationships between symptoms and diseases across a large dataset.

- Focus: Symptom–disease network analysis  
- Approach: Relational graph modeling  
- Tools: MySQL, Power BI  
- Key Contribution: Identifying high-weight connections and diagnostic patterns  

📁 Folder: graph_topology_analysis/`


## ⚙️ Technical Stack

- **Python (Pandas)** – Data cleaning and transformation  
- **MySQL** – Relational database design and query optimization  
- **Power BI** – Interactive dashboards and visual analytics  
- **SQL (Advanced Queries)** – Aggregations, joins, and analytical views

## 📂 Dataset

- Source: [Public healthcare dataset (Kaggle)](https://www.kaggle.com/datasets/kundanbedmutha/healthcare-symptomsdisease-classification-dataset)

- Structure: Symptoms, diseases, Gender,Symptom_Count,Age, Patient_ID. 

Note: Data was cleaned and normalized using Python before database integration.
