This module models relationships between symptoms and diseases using graph-based relational logic.

Key Contributions:
- Engineered a bipartite relational schema connecting symptom nodes to disease nodes
- Analyzed relational weights across 30 disease categories
- Implemented SQL-side aggregations to support scalable analytics
- Designed Power BI visualizations to identify "hub symptoms" in disease networks

Research Impact:
This system identifies high-impact symptom nodes that act as primary connectors in a medical knowledge graph.

AI Relevance:
The resulting dataset structure is optimized for future generative AI applications and automated clinical summarization.
